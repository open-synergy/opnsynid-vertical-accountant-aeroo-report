======================================
General Audit Index A.130 Aeroo Report
======================================

This module add General Audit Index A.130 report created using aeroo report

Installation
============

To install this module, you need to:

1.  Clone the branch 8.0 of the repository https://github.com/open-synergy/opnsynid-vertical-accountant-aeroo-report
2.  Add the path to this repository in your configuration (addons-path)
3.  Update the module list
4.  Go to menu *Setting -> Modules -> Local Modules*
5.  Search For *General Audit Index A.130 Aeroo Report*
6.  Install the module
